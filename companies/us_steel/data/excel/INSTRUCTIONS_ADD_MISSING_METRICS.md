# Инструкция по добавлению недостающих метрик в Excel

## Недостающие метрики

В Excel файле `us_steel_input.xlsx` отсутствуют следующие метрики, которые необходимы для построения модели:

1. **deferred_tax_assets** (DTA) - Отложенные налоговые активы
2. **deferred_tax_liabilities** (DTL) - Отложенные налоговые обязательства  
3. **lease_liab_current** - Текущие обязательства по лизингу
4. **lease_liab_noncurrent** - Долгосрочные обязательства по лизингу
5. **st_debt** - Краткосрочный долг

## Данные для добавления

Данные находятся в файле `us_steel_input_missing_metrics.csv` в том же каталоге.

## Как добавить в Excel

1. Откройте файл `us_steel_input.xlsx`
2. Перейдите на лист **history_bs**
3. Найдите строку с заголовком "metric" (обычно строка 2)
4. Добавьте следующие строки в конец таблицы:

## Альтернативный способ

Если Excel файл поврежден, можно:
1. Использовать CSV файл `us_steel_input_missing_metrics.csv` для импорта данных
2. Или восстановить файл из резервной копии `us_steel_input.backup.xlsx` (если она была создана)

## Проверка

После добавления метрик запустите:
```bash
PYTHONPATH=. python3 tools/compare_preprocessing_test_vs_prod.py --find-missing --min-year 2010
```

Этот скрипт покажет, остались ли еще недостающие метрики.
